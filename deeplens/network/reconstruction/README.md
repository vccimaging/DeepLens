# Reconstruction Package

Neural network architectures for image reconstruction and restoration in end-to-end differentiable imaging pipelines.

## Files

| File | Class | Description |
|------|-------|-------------|
| `unet.py` | `UNet` | U-Net with residual blocks (ResUNet-a based) |
| `nafnet.py` | `NAFNet` | Nonlinear Activation Free Network |
| `restormer.py` | `Restormer` | Efficient Transformer for high-resolution restoration |
| `swinir.py` | `SwinIR` | Swin Transformer for image restoration (not exported in `__init__.py`) |
